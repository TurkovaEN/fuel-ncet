# Fuel NCET (Обоснование НМЦК по топливу)

Десктопное Windows‑приложение с графическим интерфейсом для расчёта **обоснования начальной цены единицы товара** (бензин/дизель) по данным Росстата и формирования итогового документа **.docx** по шаблону Word.

## Возможности
- Графический интерфейс (без консоли).
- Автозагрузка цен с сайта **22.rosstat.gov.ru**:
  - поиск актуальной публикации *«Потребительские цены на …»*;
  - скачивание PDF по региону **Алтайский край**;
  - извлечение значений по колонке **Барнаул** для:
    - Дизельное топливо, л
    - Бензин АИ‑92, л
    - Бензин АИ‑95, л
- Поле **«Дата формирования»** (по умолчанию текущая) — можно менять; по этой дате выбираются последние доступные данные Росстата.
- Расчёт:
  - ежемесячного индекса `m` из годового индекса прогнозной инфляции (ввод вручную, напр. `104,0%`);
  - дефляторов на начало/конец поставки по принятой схеме степеней (есть ручная корректировка степеней);
  - ИПЦ на период поставки (среднее значение);
  - итоговых сумм по таблице.
- Предпросмотр таблицы в окне программы.
- Экспорт итогового документа **DOCX** по шаблону `assets/template.docx`.
- Числа в строках “прописью” (рубли/копейки) с правильными склонениями.
- Кэширование скачанного PDF Росстата в `cache/` (опционально очистка кэша при выходе).
- Кнопка **«Сброс»** — очищает все поля до начального состояния.

## Требования
### Для разработки
- Windows 10/11
- Python 3.12+
- Git (опционально)

### Для запуска готового `.exe`
- Ничего устанавливать не нужно (если сборка сделана как `--onefile`)

## Структура проекта
- `main.py` — точка входа (добавляет `src/` в `sys.path` для запуска из терминала)
- `src/fuel_ncet/` — исходный код приложения
- `assets/template.docx` — Word‑шаблон итогового документа (docxtpl)
- `assets/app.ico` — иконка приложения
- `cache/` — временные файлы (кэш PDF Росстата; может удаляться при выходе)
